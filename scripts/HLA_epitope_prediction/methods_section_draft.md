# Methods: HLA Epitope Binding Analysis

## HLA Epitope Binding Prediction Pipeline

To assess the immunological recognition potential of [organism/pathogen] proteins by human HLA molecules, we developed a computational pipeline for systematic prediction of HLA Class I and Class II epitope binding across the [organism] proteome. The pipeline is implemented in Python 3 and is available at [repository URL].

### Proteome Acquisition

Reference protein sequences for [organism] were retrieved from the NCBI Protein database using the Entrez E-utilities API [CITATION: Sayers et al., 2022] as implemented in Biopython v1.79 [CITATION: Cock et al., 2009]. Sequences were queried using the search term `"[Organism][Organism] AND RefSeq[Filter]"`, restricting retrieval to curated RefSeq entries to ensure sequence non-redundancy and annotation quality. Up to 15,000 protein sequences were downloaded in batches of 100 records in FASTA format. Peptide sequences containing ambiguous amino acid characters (X or stop codons) were excluded from all downstream analyses.

### Signal Peptide Filtering for HLA Class II Analysis

Because HLA Class II molecules present peptides derived predominantly from proteins that transit the secretory pathway, the full proteome was first screened to identify secreted proteins. Signal peptides were predicted using SignalP 6.0 [CITATION: Teufel et al., 2022, Nature Biotechnology] in fast mode with the organism type set to `other`. Proteins assigned a signal peptide (SP) classification by SignalP 6.0 were retained for Class II epitope prediction; all proteins were retained for Class I analysis irrespective of secretory signal.

### Peptide Generation

Candidate peptides were generated by sliding a fixed-length window across each protein sequence. For **HLA Class I** analysis, 9-mer peptides were generated with a step size of 3 residues, yielding approximately one-third of all possible overlapping 9-mers per protein. The pipeline supports extension to 8–11 residue lengths to capture the full Class I binding length distribution. For **HLA Class II** analysis, 15-mer peptides were generated from secreted proteins with a step size of 3 residues, consistent with the approximate length of peptides accommodated by the open-ended Class II binding groove [CITATION: Brown et al., 1993, Nature; Roche & Furuta, 2015, Annual Review of Immunology].

### HLA Binding Prediction

**Class I binding** was predicted using netMHCpan 4.2 [CITATION: Reynisson et al., 2020, Nucleic Acids Research] against a panel of 50 common HLA-A, -B, and -C alleles representative of diversity in European populations (full allele list provided in Supplementary Table X). **Class II binding** was predicted using netMHCIIpan 4.3 [CITATION: Reynisson et al., 2020, Nucleic Acids Research] against a panel of 210 HLA-DP and HLA-DQ heterodimer combinations (full list in Supplementary Table X). Both tools were run in peptide input mode (`-inptype 1`), outputting per-peptide binding scores and eluted ligand percentile ranks (%Rank).

Predictions for each allele were run independently as parallel jobs using GNU parallel [CITATION: Tange, 2011, The USENIX Magazine]. To support large-scale analyses and incremental addition of alleles, completed allele results were cached to disk; subsequent runs automatically detected and skipped previously processed alleles.

### Binding Threshold Classification

Predicted peptides were classified according to the %Rank output of netMHCpan/netMHCIIpan, which ranks binding affinity relative to a large set of random natural peptides. The following thresholds were applied:

| Binding category | HLA Class I (%Rank) | HLA Class II (%Rank) |
|------------------|--------------------|-----------------------|
| Strong binder    | < 0.5              | < 1.0                 |
| Weak binder      | 0.5 – 2.0          | 1.0 – 5.0             |
| Non-binder       | ≥ 2.0              | ≥ 5.0                 |

These thresholds are consistent with published recommendations for netMHCpan [CITATION: Reynisson et al., 2020] and have been widely applied in immunoinformatics studies [CITATION: examples as appropriate].

### Summary Statistics

For each HLA allele, we calculated the total number of predicted strong and weak binders, the percentage of all tested peptides falling within each binding category, and the number of unique proteins containing at least one strong-binding peptide. Analyses were performed using pandas v1.3 [CITATION: pandas Development Team, 2020; McKinney, 2010, Proceedings of the 9th Python in Science Conference] and results were exported as tab-separated summary files per HLA class.
